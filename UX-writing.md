# UX Writing Portfolio - Fictional Examples (Google style)

This is a **fictional UX Writing showcase**, designed according to **Google UX Writing style guidelines**.  

---

<h2>
  Onboarding and welcome screens
</h2>
<p>
  This section describes how to welcome users and introduce product benefits.
</p>
<ul>
  <li>
    <strong>Welcome message:</strong> Welcome to Medito. Start your first meditation.
  </li>
  <li>
    <strong>Benefits summary:</strong> Relax and focus better by meditating for five minutes a day.
  </li>
  <li>
    <strong>CTA buttons:</strong>
    <ul>
      <li>
        Primary: Get started
      </li>
      <li>
        Secondary: Learn more
      </li>
    </ul>
  </li>
</ul>
<h2>
  Forms and inputs
</h2>
<p>
  Use clear labels and helpful guidance for form fields.
</p>
<ul>
  <li>
    <strong>Placeholder text:</strong>
    <ul>
      <li>
        Name: Your name
      </li>
      <li>
        Email: you@example.com
      </li>
    </ul>
  </li>
  <li>
    <strong>Inline guidance:</strong>
    <ul>
      <li>
        Password: Use at least eight characters and one number.
      </li>
    </ul>
  </li>
  <li>
    <strong>Success message:</strong> Profile updated.
  </li>
</ul>
<h2>
  Empty states
</h2>
<p>
  Inform users when content is missing and provide a next step.
</p>
<ul>
  <li>
    <strong>No content:</strong> No sessions yet. Start your first meditation.
  </li>
  <li>
    <strong>No search results:</strong> No matches found. Try a different keyword.
  </li>
</ul>
<h2>
  Error messages
</h2>
<p>
  Write neutral and helpful error messages that avoid blaming the user.
</p>
<ul>
  <li>
    <strong>Form error:</strong> Enter a valid email address.
  </li>
  <li>
    <strong>Network error:</strong> Cannot connect. Check your internet connection and try again.
  </li>
  <li>
    <strong>Action blocked:</strong> Complete onboarding to use this feature.
  </li>
</ul>
